# Shaarli-origr3n

A clean, opinionated [Shaarli](https://github.com/shaarli/Shaarli) theme built on the **gr3n design language**:

- **Deep Forest** color palette (`#3d8e12`)
- **Quicksand** variable font
- **Dark / Light toggle** with `localStorage` persistence and `prefers-color-scheme` fallback
- Minimal, readable link layout

> Compatible with Shaarli v0.13+

## Installation

1. Download or clone this repository
2. Copy the `origr3n/` directory into your Shaarli installation's `tpl/` directory:
   ```
   cp -r origr3n/ /path/to/shaarli/tpl/
   ```
3. In Shaarli: **Tools → Change theme → origr3n**

---

## Font

origr3n uses [Quicksand](https://fonts.google.com/specimen/Quicksand) served from `gruenheit.de/fonts/`.  
If you prefer self-hosting the font, download the variable font files and update `css/origr3n.css` accordingly.
