Was ist Shaarli?
Die Bookmark-Plattform für Individualisten

Shaarli läuft auf dem eigenen Server. Keine Datenbank, keine Cloud, keine Registrierung. Das Backend ist eine PHP-Datei, die Daten liegen in einem einzigen Flat-File. Wer sichern will, kopiert die Datei. Wer umzieht, überträgt sie.

Was es von Cloud-Diensten unterscheidet: die Daten bleiben da, wo man sie hinlegt — auf dem eigenen Server, ohne Abhängigkeit von US-amerikanischen Anbietern. Kein Drittanbieter dem man vertrauen muss. Kein Dienst der morgen eingestellt werden kann. Das Projekt läuft seit über zehn Jahren — nicht weil jemand Geld damit verdient, sondern weil es gebraucht wird.

Lesezeichen speichern
URLs mit Titel, Beschreibung und Tags in Sekunden erfassen. Jeder Link einzeln als privat oder öffentlich markierbar — kein Alles-oder-Nichts.
Flexibles Tag-System
Tags statt starrer Ordner-Hierarchien. Tag-Cloud zeigt Schwerpunkte auf einen Blick. Filterbar und kombinierbar — mehrere Tags und ein Suchbegriff gleichzeitig.
Volltextsuche
Durchsucht Titel, Beschreibung und Tags simultan. Treffer sofort, keine Ladezeiten. Ergebnisse zeigen genau, was gesucht wird — auch bei langer Linkliste.
Privat oder öffentlich
Jeder Link für sich: privat oder öffentlich. Private Links sind nur nach Login sichtbar — ideal als persönliches Notizbuch neben dem öffentlichen Feed.
Bildwand
Alle Links mit Vorschaubild als visuelle Galerie. Ideal für Inspirationsquellen, Design-Referenzen oder alles, wo das Bild mehr sagt als die URL.
Tagesansicht
Tagesrückblick aller gespeicherten Links. Wer täglich hinzufügt, bekommt hier eine saubere Übersicht nach Datum.
RSS/Atom-Feeds
Jede Tag-Ansicht, jede Suche und die gesamte Liste als Feed abonnieren. Feedreader, Newsletter-Tools und Aggregatoren lassen sich direkt anbinden.
REST-API
Vollständige CRUD-API zum programmatischen Verwalten von Links. Ideal für Browser-Erweiterungen, CLI-Shortcuts und Automatisierungs-Workflows — ohne Abhängigkeit vom UI.
Plugin-System
Markdown-Rendering, QR-Code-Generator, Thumbnail-Vorschau und mehr. Offizielle und Community-Plugins erweitern das Verhalten — Shaarli selbst bleibt unangetastet.
Import & Export
Netscape Bookmark Format für direkten Import aus Chrome und Firefox. Wer von einem anderen Dienst kommt, verliert dabei keine Daten.
Keine Datenbank
Alle Daten in einem einzigen Flat-File. Kein MySQL, kein Redis, kein Overhead. Eine Datei sichern, eine Datei übertragen — mehr nicht.
Selbst hosten, Daten behalten
Eigener Server, eigene Regeln — kein US-Anbieter, kein Drittdienst. Die Lesezeichen verlassen den eigenen Server nicht. Aus europäischer Sicht: volle Datensouveränität, ohne DSGVO-Kompromisse. Läuft auf jedem PHP-fähigen Webserver ab PHP 7.4.

Wo origr3n tatsächlich etwas ändert
Nicht alles davon fällt sofort auf. Manches merkt man erst, wenn man kurz ohne auskommt.
Deep Forest Dark Theme
Kein invertiertes Farbschema. Die Kontraste sind dort, wo sie hingehören — Text lesbar, UI-Elemente erkennbar, ohne dass die Augen nach einer Weile protestieren.
Hell/Dunkel-Umschalter
Manuelles Umschalten oder Anlehnen an die System-Einstellung. Präferenz wird im Browser gespeichert — beim nächsten Besuch sofort richtig.
Suchoverlay + /-Shortcut
Taste / öffnet das Such-Overlay von jeder Stelle der Seite. Kein Scrollen, keine Mausklicks.
Tag-Autocomplete
Awesomplete schlägt beim Tippen passende Tags aus der eigenen Bibliothek vor. Kein „linux" und „Linux" und „linux-2" mehr — die Sammlung bleibt so wie man sie gemeint hat.
Bildwand-Optimierung
Exakt 8 Kacheln pro Reihe, zentrierte letzte Reihe, gleichmäßige Abstände — auf jeder Bildschirmbreite ohne manuelle Einstellungen.
Status-Badges im Footer
Privat- und Gepinnt-Status stehen im Footer jeder Karte, nicht neben dem Titel. Wer es sehen will, sieht es. Wer nur liest, wird nicht davon gestört.
Direktzugriff — Link hinzufügen
Der + Button ist auf der Linkliste immer sichtbar, ohne zu stören. Neuen Link hinzufügen ohne Umwege über das Menü — direkt aus der Übersicht heraus.
Keine Suchergebnisse — sauber kommuniziert
Keine Treffer — kontextueller Hinweis und Illustration statt leerer Seite.
i18n — Spracherkennung
Dynamische Texte folgen der Browser-Spracheinstellung. Deutsch und Englisch sind eingebaut, weitere Sprachen leicht erweiterbar — ohne Theme-Umbau.
Quicksand Typografie
Quicksand ist die Lieblingsschrift des Theme-Autors — und bleibt es. Lokal eingebettet unter der SIL Open Font License — kein CDN, keine externen Anfragen.
Zeitstrahl-Ansicht
Ein Uhr-Icon im Header schaltet Datums-Trennlinien zwischen den Link-Karten ein — chronologische Orientierung auf einen Blick. Die Einstellung ist persistent.
